# Poster

Отправка request и получение response

![](./assets/images/poster.jpg)

## Usage

```bash
go run poster.go --requests <путь> --responses <путь>
```

## Structure
```
Poster/
|-- poster.go
|-- requests/
|   |-- request1.json
|   |-- request2.json
|   └-- ...
|-- responses/
|   |-- response1.json
|   |-- response2.json
|   └-- ...
|-- internal/
```
